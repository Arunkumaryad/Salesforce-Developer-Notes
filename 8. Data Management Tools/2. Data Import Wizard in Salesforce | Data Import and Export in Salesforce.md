# Data Import Wizard in Salesforce | Data Import and Export in Salesforce

## Introduction

Salesforce provides powerful tools to import, update, export, and manage data.

One of the easiest tools for importing data into Salesforce is the **Data Import Wizard**.

The Data Import Wizard is a browser-based tool that allows administrators and users to import records without installing any software.

This guide explains:

- What is Data Import Wizard
- Features of Data Import Wizard
- Supported Operations
- Difference Between Data Import Wizard and Data Loader
- Step-by-Step Import Process
- Real Example (Lead Import)

- Interview Questions

# What is Data Import Wizard?

## Definition

**Data Import Wizard** is a Salesforce web-based tool that allows users to import data into Salesforce using CSV files.

According to Salesforce:

```text
Data Import Wizard is an in-browser wizard
that imports your organization's data.
```

It provides a simple user interface for importing records and is ideal for small to medium-sized data loads.

---

# Why Use Data Import Wizard?

The Data Import Wizard is useful when:

```text
Importing new records
Updating existing records
Upserting records
Importing data without installing software
Handling up to 50,000 records
```

# Difference Between Data Import Wizard and Data Loader

| Data Import Wizard | Data Loader |
|-------------------|-------------|
| It is an in-browser wizard that imports your organization's data. | It is a desktop application for bulk imports and exports. |
| Supports up to 50,000 records. | Supports up to 5 million records (50 lakh). |
| Works with specific objects such as Accounts, Contacts, Leads, Solutions, and Custom Objects. | Works with all standard and custom objects. |
| Does not support Opportunities. | Supports Opportunities. |
| Does not provide detailed logs. | Provides detailed logs. |
| Does not support scheduling. | Supports scheduled imports and exports. |
| Does not support Delete operation. | Supports Delete and Hard Delete operations. |
| Duplicate records can be ignored during import. | Duplicate records cannot be automatically ignored. |

# Steps to Use Data Import Wizard

Salesforce Data Import Wizard follows a simple step-by-step process.

---

# Step 1: Prepare the Data File (.csv)

Create a CSV file containing the records you want to import.

Example:

| First Name | Last Name | Email | Company |
|------------|-----------|--------|---------|
| Arun | Kumar | arun@gmail.com | ABC Pvt Ltd |
| Rahul | Singh | rahul@gmail.com | XYZ Ltd |

Save the file as:

```text
leads_import.csv
```

---

## Important Guidelines

```text
Use CSV format
Keep field names clear
Remove duplicate records
Validate mandatory fields
```

---

# Step 2: Launch Data Import Wizard

Navigate to:

```text
Setup
→ Search "Data Import Wizard"
→ Data Import Wizard
```

Click:

```text
Launch Wizard
```

---

# Navigation Path

```text
Setup
→ Data Import Wizard
→ Launch Wizard
```

---

# Step 3: Select Standard or Custom Object

After launching the wizard, Salesforce displays available objects.

Choose:

```text
Standard Objects
```

or

```text
Custom Objects
```

Example:

```text
Leads
Accounts
Contacts
Custom Objects
```

For this example select:

```text
Leads
```

---

# Step 4: Select the Activity to be Performed

Salesforce provides three options:

```text
Add New Records
Update Existing Records
Add New and Update Existing Records
```

---

## Add New Records

Creates new records only.

```text
Insert Operation
```

---

## Update Existing Records

Updates records already present in Salesforce.

```text
Update Operation
```

---

## Add New and Update Existing Records

Creates new records and updates existing records.

```text
Upsert Operation
```

---

# Example

Select:

```text
Add New Records
```

because we are importing new Leads.

---

# Step 5: Upload CSV File

Upload the prepared CSV file.

Example:

```text
leads_import.csv
```

Salesforce provides options such as:

```text
Character Encoding
Field Separator
CSV Format
```

---

## Example Configuration

```text
Character Code:
ISO-8859-1

Values Separated By:
Comma
```

Click:

```text
Next
```

---

# Step 6: Map CSV Fields with Salesforce Fields

Salesforce automatically maps fields if the CSV headers match Salesforce field names.

Example:

| CSV Header | Salesforce Field |
|------------|------------------|
| First Name | First Name |
| Last Name | Last Name |
| Email | Email |
| Company | Company |

---

## Auto Mapping Example

```text
CSV Column           Salesforce Field
--------------------------------------
First Name      →    First Name
Last Name       →    Last Name
Email           →    Email
Lead Source     →    Lead Source
Industry        →    Industry
```

---

## Manual Mapping

If a field is not mapped automatically:

```text
Click Change
Select Salesforce Field
Save Mapping
```

---

## Important Note

```text
Unmapped fields
will not be imported.
```

Always verify mapping before proceeding.

---

# Step 7: Perform the Import

Salesforce displays a review screen.

Review:

```text

Operation Type
Uploaded File
Mapped Fields
Unmapped Fields
```

Example:

```text
Object: Leads
Operation: Add New Records
Mapped Fields: 14
Unmapped Fields: 0
```

Click:

```text
Start Import
```

# Summary

Data Import Wizard is a browser-based Salesforce tool used to import, update, and upsert data using CSV files. It supports up to 50,000 records and is ideal for administrators who need a simple, user-friendly way to load data into Salesforce. The process involves preparing a CSV file, launching the wizard, selecting an object, choosing an operation, uploading the file, mapping fields, starting the import, monitoring job status, and correcting any errors. For larger data volumes or advanced operations such as Delete, Hard Delete, Export, and Scheduling, Salesforce recommends using Data Loader.
