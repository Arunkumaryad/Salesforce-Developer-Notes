# Salesforce Data Export & Scheduled Export Wizard | How Can You Export Data in Salesforce?

## Introduction

Data is one of the most valuable assets in Salesforce. To protect business information and maintain backups, Salesforce provides a built-in feature called **Data Export**.

The Data Export Wizard allows administrators to export Salesforce data manually or schedule automatic exports at regular intervals.

This guide explains:

- What is Data Export?
- Why Data Export is Important
- Features of Data Export
- Manual Data Export
- Scheduled Data Export
- Export File Formats
- Downloading Exported Files
- Best Practices
- Interview Questions

---

# What is Data Export in Salesforce?

## Definition

According to Salesforce:

> **Data Export lets you prepare a copy of all your data in salesforce.com. From this page you can start the export process manually or schedule it to run automatically. When an export is ready for download you will receive an email containing a link that allows you to download the file(s). The export files are also available on this page for 48 hours, after which time they are deleted.**

Data Export is primarily used for:

- Backup purposes
- Data migration
- Reporting and analysis
- Disaster recovery
- Data archival

---

# Why Use Data Export?

Organizations use Data Export to:

✅ Create regular backups

✅ Protect business-critical data

✅ Export data before major deployments

✅ Move data to another Salesforce org

✅ Perform audits and compliance checks

✅ Analyze Salesforce data externally

---

# Features of Data Export

### Manual Export

Allows users to export data immediately.

### Scheduled Export

Automatically exports data at predefined intervals.

### ZIP File Download

Exported data is delivered as ZIP files.

### CSV Format

Records are exported in CSV format.

### Email Notification

Salesforce sends an email when the export is ready.

### Supports Standard and Custom Objects

Exports data from:

- Standard Objects
- Custom Objects

---

# Data Export vs Data Loader Export

| Feature | Data Export Wizard | Data Loader Export |
|----------|-------------------|-------------------|
| Purpose | Full Org Backup | Export Specific Objects |
| Export Multiple Objects | ✅ Yes | ❌ One Object at a Time |
| Scheduled Export | ✅ Yes | Limited |
| CSV Output | ✅ Yes | ✅ Yes |
| ZIP File | ✅ Yes | ❌ No |
| Easy to Use | ✅ Yes | Moderate |
| Setup Required | ❌ No | Requires Installation |

# Manual Data Export

Manual export creates a backup immediately.

---

## Step 1: Click Export Now

From the Monthly Export Service page:

```text
Export Now
```

---

## Step 2: Configure Export Settings

Salesforce opens the export configuration page.

### Export File Encoding

Example:

```text
ISO-8859-1
```

Used to define character encoding for exported files.

---

### Include Images, Documents and Attachments

Select if you want to export:

```text
Attachments
Documents
Images
```

---

### Include Salesforce Files and CRM Content

Select if exporting:

```text
Salesforce Files
Content Documents
CRM Content Versions
```

---

### Replace Carriage Returns With Spaces

Recommended for cleaner CSV files.

---

# Step 3: Select Objects to Export

Under **Exported Data**, choose:

```text
Include All Data
```

or select specific objects.

Examples:

```text
Account
Contact
Lead
Opportunity
Case
Custom Objects
```

Example custom objects:

```text
Student1__c
Teacher__c
Book__c
Project__c
```

---

# Step 4: Start Export

Click:

```text
Start Export
```

Salesforce begins preparing the export.

---

# Export Processing

After clicking Start Export, Salesforce displays:

```text
A data export is currently in progress for your organization.
```

You will also see:

```text
Your export has been queued.
You will receive an email notification when it is completed.
```

---

# Email Notification

When export processing completes, Salesforce sends an email.

Example message:

```text
Your Organization Data Export has completed.
```

The email contains a download link.

Important:

```text
Download within 48 hours.
```

After 48 hours the export file is removed.

---

# Download Export Files

Open the email link.

Salesforce displays available export files.

Example:

```text
WE_00DgL000004VLraUAG_1.ZIP
```

Click:

```text
Download
```

---

# Extract ZIP File

Downloaded file is usually:

```text

```

Extract the ZIP file.

# Export File Availability

Important Salesforce rule:

```text
Export files remain available for 48 hours.
```

After 48 hours:

```text
Files are automatically deleted.
```

Always download backups immediately.

# Summary

Salesforce Data Export is a built-in backup tool that allows administrators to export data manually or schedule automatic exports. The exported data is provided in ZIP files containing CSV records for standard and custom objects. After an export is completed, Salesforce sends an email notification with a download link. Export files remain available for 48 hours, making Data Export an essential feature for backup, migration, compliance, and disaster recovery purposes.
