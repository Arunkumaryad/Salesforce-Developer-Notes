# What is Data Management?

## Definition

Data Management is the process of importing, updating, deleting, exporting, and maintaining data within Salesforce.

It helps organizations keep their Salesforce data accurate, clean, and up to date.

# Salesforce Data Management Tools

Salesforce provides two primary tools for data management:

```text
1. Data Import Wizard
2. Data Loader
```

---

# Data Import Wizard

## Definition

Data Import Wizard is a Salesforce web-based tool that allows users to import data into Salesforce without installing any software.

It is beginner-friendly and suitable for small to medium-sized data imports.

---

## Features of Data Import Wizard

```text
Browser Based Tool
Easy to Use
Supports Duplicate Checking
No Installation Required
Suitable for Small Data Volumes
```

---

## Supported Operations

```text
Import
Insert
Update
Upsert
```

---

## Supported Objects

Examples:

```text
Accounts
Contacts
Leads
Solutions
Campaign Members
Custom Objects
```

---

## Limit

```text
Up to 50,000 Records
```

per import operation.

---

## Navigation

Navigate to:

```text
Setup
→ Data Import Wizard
→ Launch Wizard
```

---

# Data Loader

## Definition

Data Loader is a Salesforce client application used for bulk importing, exporting, updating, deleting, and migrating Salesforce data.

It is designed for handling large volumes of records.

---

## Features of Data Loader

```text
Desktop Application
Supports Bulk Operations
Supports Scheduling
Provides Detailed Logs
Works With All Objects
```

---

## Supported Operations

```text
Insert
Update
Upsert
Delete
Hard Delete
Export
Export All
```

---

## Supported Objects

```text
All Standard Objects
All Custom Objects
```

Examples:

```text
Accounts
Contacts
Leads
Opportunities
Cases
Custom Objects
```

---

## Record Capacity

```text
Up to 5 Million Records
(50 Lakh Records)
```

depending on Salesforce limits and API usage.

---

## Navigation

Navigate to:

```text
Setup
→ Data Loader
→ Download Data Loader
```

Install the application and connect it with Salesforce.

---

# Difference Between Data Import Wizard and Data Loader

| Feature | Data Import Wizard | Data Loader |
|----------|------------------|-------------|
| Type | In-browser Wizard | Desktop Application |
| Record Limit | Up to 50,000 Records | Up to 5 Million Records |
| Installation | Not Required | Required |
| Supported Objects | Specific Standard & Custom Objects | All Standard & Custom Objects |
| Opportunity Support | ❌ Not Supported | ✅ Supported |
| Detailed Logs | ❌ No | ✅ Yes |
| Scheduling | ❌ No | ✅ Yes |
| Delete Operation | ❌ No | ✅ Yes |
| Hard Delete | ❌ No | ✅ Yes |
| Export Data | Limited | Full Export Support |
| Duplicate Handling | Can Ignore Duplicates | Cannot Automatically Ignore Duplicates |
| Best For | Small Imports | Bulk Data Operations |

# Detailed Comparison

## Data Import Wizard

### 1. Browser-Based Tool

```text
No installation required.
Runs directly from Salesforce Setup.
```

### 2. Record Limit

```text

```

### 4. Logging


### 5. Scheduling

```text

```

Cannot automate nightly imports.

---

### 6. Delete Operation

```text
Not supported.
```

---

### 7. Duplicate Handling

During import:

```text

```

# When to Use Data Import Wizard

Use Data Import Wizard when:

```text
Data Volume is Small
Less Than 50,000 Records
No Installation Needed
Simple Import Required
Duplicate Detection Needed
```

Example:

```text
Import 5,000 Leads
Import 10,000 Contacts
```

---

# When to Use Data Loader

Use Data Loader when:

```text
Large Data Volumes
Bulk Updates
Bulk Deletes
Opportunity Imports
Data Migration Projects
Scheduled Imports
Need Detailed Logs
```

Example:

```text
Import 1 Million Accounts
Update 500,000 Contacts
Delete 200,000 Records
```

# Summary

Salesforce provides two major data management tools: Data Import Wizard and Data Loader. Data Import Wizard is a browser-based tool designed for simple imports of up to 50,000 records and supports duplicate checking. Data Loader is a desktop application designed for large-scale data operations, supporting up to 5 million records, detailed logs, scheduling, exports, deletes, and all standard and custom objects. For small imports, Data Import Wizard is preferred, while Data Loader is the recommended solution for enterprise-level data migration and bulk processing.
