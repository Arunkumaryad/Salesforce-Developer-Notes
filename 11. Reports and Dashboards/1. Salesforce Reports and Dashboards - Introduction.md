# Salesforce Reports and Dashboards - Introduction

## Introduction

Reports and Dashboards are among the most powerful analytics features in Salesforce. They help users analyze data, track business performance, and make informed decisions.

A **Report** helps organize and display data from Salesforce records, while a **Dashboard** provides a visual representation of report data using charts, graphs, tables, and metrics.

---

# What are Reports?

## Definition

A **Report** is a collection of information and an organized presentation of data about a particular object (or with its related objects) under certain criteria.

### In Simple Words

A report allows you to:

- Retrieve data from Salesforce objects
- Filter records based on conditions
- Organize information in a meaningful format
- Analyze business data

### Example

Suppose we have a Student object.

A report can display:

```text
All Students
```

or

```text
Students with Score > 80
```

or

```text
Students from Computer Science Department
```

---

# What is a Dashboard?

## Definition

A **Dashboard** is the graphical representation of the data generated by a report (or multiple reports) at a certain point in time.

It is a snapshot of data.

To obtain the latest data, dashboards must be refreshed either:

- Manually
- On a scheduled basis

### In Simple Words

A dashboard converts report data into visual components such as:

- Charts
- Graphs
- Gauges
- Tables
- Metrics

---

# Report vs Dashboard

| Report | Dashboard |
|----------|----------|
| Displays data in tabular format | Displays data visually |
| Used for detailed analysis | Used for quick insights |
| Source of data | Uses report data |
| Shows records | Shows charts and summaries |
| Can exist independently | Requires reports |

**Report Builder** is a tool for accessing your data quickly and comprehensively.

Use it to:

- Add filters

- Generate summaries

# Types of Salesforce Reports

Salesforce provides four report formats.

## 1. Tabular Report

The simplest report format.

Displays data in rows and columns.

Example:

```text
List of Students
```

---

## 2. Summary Report

Allows grouping of records.

Example:

```text
Students grouped by College
```

---

## 3. Matrix Report

Groups records by rows and columns.

Example:

```text
Students grouped by College and Branch
```

---

## 4. Joined Report

Combines multiple report types into one report.

Example:

```text
Students and Teachers data together
```

---

# How Reports and Dashboards Work Together

```text
Salesforce Records
          ↓
        Report
          ↓
      Dashboard
          ↓
 Business Insights
```

### Flow Explanation

1. Data is stored in Salesforce objects.
2. Reports retrieve and organize data.
3. Dashboards visualize report data.
4. Users analyze business performance.

# Interview Questions

## What is a Report in Salesforce?

A Report is a collection of information and an organized presentation of data about a particular object (or related objects) under certain criteria.

---

## What is a Dashboard in Salesforce?

A Dashboard is the graphical representation of data generated by one or more reports at a specific point in time.

---

## What is Report Builder?

Report Builder is a tool used to create, customize, and manage reports in Salesforce.

---

## Can a Dashboard Exist Without a Report?

No.

Dashboards require one or more source reports.

---

## Why is a Dashboard Called a Snapshot?

Because it shows report data at a particular moment in time and requires refreshing to display the latest information.

---

## What are the Four Types of Reports?

1. Tabular Report
2. Summary Report
3. Matrix Report
4. Joined Report

# Summary

Reports and Dashboards are Salesforce analytics tools used to analyze and visualize business data. A **Report** is a collection and organized presentation of data based on specific criteria, while a **Dashboard** is a graphical representation of data generated by one or more reports. Report Builder helps create and customize reports efficiently. Together, Reports and Dashboards provide powerful insights, helping organizations monitor performance, track key metrics, and make data-driven decisions.
