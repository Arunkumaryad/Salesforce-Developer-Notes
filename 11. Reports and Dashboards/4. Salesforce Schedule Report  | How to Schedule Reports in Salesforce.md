# Salesforce Schedule Report | How to Schedule Reports in Salesforce

## Overview

### What is a Scheduled Report?

Schedule a report to run daily, weekly, or monthly.

An HTML version of the report can be sent by email to users in your organization.

### Use

To get regular periodic updates in terms of reports without manually running the report every time.

---

# Why Schedule Reports?

Scheduled Reports help users:

- Automatically receive report results via email.
- Monitor important business data regularly.
- Track KPIs and performance metrics.
- Reduce manual effort.
- Stay updated without logging into Salesforce.

### Example

A Sales Manager wants to receive an Account Revenue Report every morning at 8:00 AM.

Instead of manually opening and running the report daily, Salesforce can automatically run the report and send the results by email.

---

# Prerequisites

Before scheduling a report:

- A report must already exist.
- The report should be saved.
- You must have permission to subscribe to reports.
- Email Deliverability should be enabled.

---

# Navigation

```text
App Launcher
    ↓
Reports
```

Open the report that you want to schedule.

Example:

```text
New Test Account Reports
```

---

# Step 1: Open the Report

Navigate to:

```text
Reports Tab
```

Locate your report.

Example:

```text
New Test Account Reports
```

Click the dropdown arrow (▼) beside the report.

---

# Step 2: Subscribe to the Report

From the dropdown menu select:

```text
Subscribe
```

Salesforce opens the:

```text
Edit Subscription
```

window.

---

# Step 3: Configure Schedule Settings

## Frequency Options

Salesforce provides three scheduling frequencies:

### Daily

```text
Runs every day
```

Example:

```text
2:00 AM daily
```

---

### Weekly

```text
Runs once every week
```

Example:

```text
Every Monday at 9:00 AM
```

---

### Monthly

```text
Runs once every month
```

Example:

```text
1st Day of Every Month
```

---

## Select Time

Choose the execution time.

Example:

```text
2:00 AM
```

Salesforce will automatically run the report at the selected time.

---

# Step 4: Configure Attachment

You can send the report as an attachment.

Click:

```text
Attach File
```

Supported report formats include:

- Excel (.xlsx)
- CSV
- Formatted Report
- HTML Email Version

---

# Step 5: Select Recipients

Under:

```text
Recipients
```

Choose who should receive the report.

Options:

### Me

```text
Current Logged-in User
```

---

### Other Users

Click:

```text
Edit Recipients
```

Add:

- Users
- Roles
- Public Groups

---

## Important Note

Recipients see the same report data as the person running the report.

---

# Step 6: Run Report As

Salesforce allows two options.

### Me

```text
Report runs using my access and permissions.
```

---

### Another Person

```text
Report runs using another user's permissions.
```

Search and select the desired user.

Example:

```text
Arun Kumar Yadav
```

---

# Step 7: Add Conditions (Optional)

Conditions allow Salesforce to send notifications only when specific criteria are met.

Enable:

```text
Add conditions to this report
```

---

## Example Condition

### Aggregate

```text
Sum of Annual Revenue
```

### Operator

```text
Equals
```

### Value

```text
100
```

Meaning:

```text
Send notification only when
Sum of Annual Revenue = 100
```

---

## Additional Condition Options

Operators include:

- Equals
- Not Equal To
- Greater Than
- Less Than
- Greater Than or Equal To
- Less Than or Equal To

You can click:

```text
+ Add Condition
```

to add multiple conditions.

---

# Step 8: Save Subscription

After configuring all settings click:

```text
Save
```

Salesforce creates the report subscription.

---

# Verify Scheduled Report

To verify that the report has been scheduled:

Navigate to:

```text
Setup
    ↓
Scheduled Jobs
```

Search:

```text
Scheduled Jobs
```

Open:

```text
Scheduled Jobs
```

---

# View Scheduled Report Job

You will see an entry similar to:

| Field | Example |
|---------|---------|
| Job Type | Reporting Notification |
| Submitted By | Arun Kumar Yadav |
| Next Scheduled Run | 2:00 AM |
| Status | Scheduled |

Example:

```text
Reporting Notification
```

This confirms that Salesforce has successfully scheduled the report.

---

# Delete a Scheduled Report

Navigate to:

```text
Setup
    ↓
Scheduled Jobs
```

Locate:

```text
Reporting Notification
```

Click:

```text
Del
```

The scheduled report will be removed.

# Limitations

- Scheduled reports only run at predefined intervals.
- Users must have access to the report.
- Recipients see data based on the running user's permissions.
- Organization limits apply to report subscriptions.

# Interview Questions

## What is a Scheduled Report in Salesforce?

A Scheduled Report is a report that runs automatically at a specified time and sends report results to selected users via email.

---

## What frequencies are available for Scheduled Reports?

Salesforce supports:

1. Daily
2. Weekly
3. Monthly

---

## What is the difference between Running User and Recipient?

The Running User determines what data the report can access, while Recipients receive the report output.

---

## Can we add conditions to Scheduled Reports?

Yes. Salesforce allows condition-based notifications using aggregates and operators.

---

## Where can we verify scheduled reports?

Navigate to:

```text
Setup
→ Scheduled Jobs
```

and check for:

```text
Reporting Notification
```

entries.

---

## Can we stop a scheduled report?

Yes.

Navigate to:

```text
Setup
→ Scheduled Jobs
```

and delete the corresponding Reporting Notification job.

# Real-World Scenario

## Requirement

The management team wants a daily email containing Account Revenue data at 2:00 AM.

## Solution

1. Create the report.
2. Open the report.
3. Click **Subscribe**.
4. Select **Daily**.
5. Set time to **2:00 AM**.
6. Add recipients.
7. Save.

Salesforce will automatically run the report and email the results every day.
