# How to Create Salesforce Developer Org

## Introduction

A Salesforce Developer Org is a free environment provided by Salesforce for learning, development, testing, and practice purposes.

It helps developers and administrators:
- Learn Salesforce
- Build applications
- Practice Apex and LWC
- Test features
- Explore Salesforce functionalities

---

# What is Salesforce Developer Org?

A Salesforce Developer Org is a free development environment provided by Salesforce.

It is mainly used for:
- Development
- Testing
- Learning
- Customization
- Practice

Developer Org does not contain live customer data.

---

# Create Salesforce Developer Account

## Official Signup URL

To create account:

```text
developer.salesforce.com/signup
```

---

# Salesforce Login URL

To log in:

```text
login.salesforce.com
```

---

# Step-by-Step Process to Create Salesforce Developer Org

## Step 1: Open Signup Page

Open the official Salesforce Developer signup page:

```text
https://developer.salesforce.com/signup
```

---

# Step 2: Fill Required Details

Enter the following information:

- First Name
- Last Name
- Email Address
- Job Role
- Company Name
- Country

---

# Step 3: Accept Terms and Conditions

Check:
- Terms and Conditions
- Privacy Policy

Then click:
```text
Sign Me Up
```

---

# Step 4: Verify Your Email

Salesforce will send a verification email.

Open your email and click:
```text
Verify Account
```

---

# Step 5: Set Password

Create:
- Username
- Password
- Security Question

---

# Step 6: Login to Salesforce

Open:

```text
https://login.salesforce.com
```

Enter:
- Username
- Password

Now your Salesforce Developer Org is ready.

# Difference Between Production Org and Developer Org

| Production Org | Developer Org |
|----------------|----------------|
| Used for live business data | Used for learning and testing |
| Contains real customer data | Contains test data |
| Used in actual business operations | Used for development and practice |
| Limited experimentation | Safe for testing |

- Do not use it for production business operations.
- You can create multiple Developer Orgs for practice.
