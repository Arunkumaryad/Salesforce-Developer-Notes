# What is Apex? | Features of Apex Programming Language

## Introduction

Apex is a strongly typed, object-oriented programming language developed by Salesforce.

It is mainly used for:
- Custom business logic
- Automation
- Database operations
- Integrations
- Backend development in Salesforce

Apex code runs on the Force.com platform.

---

# What is Apex?

Apex is a proprietary programming language developed by Salesforce Inc.

It allows developers to:
- Execute flow and transaction control statements
- Work with Salesforce database
- Create custom business logic
- Build scalable applications

Apex is similar to Java in syntax and supports object-oriented programming concepts.

---

# Features of Apex Programming Language

## 1. Apex language has Java like syntax.

Apex syntax is very similar to Java programming language.

Developers who know Java can easily learn Apex.

---

## 2. Apex is Object Oriented Programming Language.

Apex supports:
- Classes
- Objects
- Inheritance
- Polymorphism
- Encapsulation
- Abstraction

This makes Apex powerful and reusable.

---

## 3. Apex is strongly typed language.

A strongly typed language is a programming language that has strict restrictions over types of values and variables.

If there is mismatch between data types during:
- Variable assignment
- Function call
- Function returning value
- Parameters passing to function

then the system may generate:
- Error
- Exception

---

# Strongly Typed Language

Programming languages which has strict restrictions over types of values and variables are called Strongly Typed Language.

Examples:
- Java
- Ruby
- Smalltalk
- Python

---

## Important Point

In Java, types are associated with variable while in Python, types are associated with values.

---

## Typing Errors

- In Java, we get typing errors at compile time.
- In Python and Ruby, we get typing errors at run time.

---

# Weakly Typed Language

A weakly typed language has looser typing rules and may produce unpredictable or even erroneous results.

It may perform implicit type conversion at runtime.

Examples:
- JavaScript
- Perl

---

## Example

JavaScript allows operations between variables of different data types.

That is why JavaScript is considered weakly typed language.

---

## 4. Apex is proprietary language developed by Salesforce Inc and its code run on force.com platform.

Apex is owned and managed by Salesforce.

All Apex code executes on:
```text
Force.com Platform
```

---

## 5. Apex is a compiled language.

Apex code is compiled before execution.

Compilation helps:
- Detect errors
- Improve performance
- Maintain code quality

---

## 6. Apex is upgraded automatically.

Salesforce automatically upgrades Apex during Salesforce releases.

Developers do not need to manually upgrade the language.

---

## 7. Apex is integrated with database.

Apex directly works with Salesforce database.

Developers can:
- Query data
- Insert records
- Update records
- Delete records

using:
- SOQL
- SOSL
- DML operations

---

## 8. Apex is easy to test. It has its own testing platform.

Salesforce provides:
- Test Classes
- Code Coverage
- Testing Framework

to test Apex applications.

Testing is mandatory before deployment.

---

## 9. Apex has multi tenant environment.

Salesforce is a multi-tenant platform.

Multiple organizations share:
- Resources
- Infrastructure
- Platform

while keeping their data secure and isolated.

---

## 10. Apex code is saved with its version.

Salesforce maintains versioning for Apex code.

Different classes can use different API versions.

This helps maintain backward compatibility.

---

## 11. Apex is case insensitive language.

Apex treats uppercase and lowercase letters similarly.

Example:

```apex
String name;
STRING Name;
```

Both are treated similarly in Apex.

# Difference Between Strongly Typed and Weakly Typed Language

| Strongly Typed Language | Weakly Typed Language |
|-------------------------|-----------------------|
| Strict typing rules | Looser typing rules |
| Type mismatch gives errors | Allows implicit conversion |
| More secure and reliable | More flexible |
| Example: Java | Example: JavaScript |
