# How to Write Your First Apex Program

## Introduction

Apex is a strongly typed, object-oriented programming language provided by Salesforce.

It is mainly used for:
- Custom business logic
- Automation
- Database operations
- Integrations
- Backend development in Salesforce

This tutorial explains how to write and execute your first Apex program in Salesforce.

---

# What is Apex?

Apex is Salesforce’s proprietary programming language that runs on the Salesforce platform.

It is similar to Java and is used to:
- Write custom logic
- Create triggers
- Develop integrations
- Perform database operations

---

# First Apex Program

## Hello World Program

```apex
System.debug('Hello World');
```

---

# Explanation of the Program

## System.debug()

`System.debug()` is used to print output or debug messages in Salesforce logs.

It helps developers:
- Check program execution
- Debug code
- View variable values
- Identify errors

---

# Meaning of Hello World Program

```apex
System.debug('Hello World');
```

This statement prints:
```text
Hello World
```

inside the Salesforce Debug Log.

---

# Step-by-Step Process to Execute Apex Program

## Step 1: Login to Salesforce

Open:

```text
https://login.salesforce.com
```

Enter:
- Username
- Password

---

# Step 2: Open Developer Console

After login:

Click:
```text
Profile Icon
→ Developer Console
```

Developer Console is used to:
- Write Apex code
- Execute code
- Debug applications
- View logs

---

# Step 3: Open Execute Anonymous Window

Inside Developer Console:

Click:
```text
Debug
→ Open Execute Anonymous Window
```

Shortcut:
```text
Ctrl + E
```

---

# Step 4: Write Apex Code

Enter the following code:

```apex
System.debug('Hello World');
```

---

# Step 5: Execute the Program

Click:
```text
Execute
```

Salesforce will execute the Apex code.

---

# Step 6: View Output in Logs

To see output:

Click:
```text
Logs
```

Search for:
```text
Hello World
```

You will see the debug output in the execution log.

---

# Output

```text
Hello World
```

---

# Why System.debug() is Important

`System.debug()` is very useful for:

It is one of the most commonly used methods in Apex development.
