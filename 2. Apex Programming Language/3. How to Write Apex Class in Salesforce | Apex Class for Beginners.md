# How to Write Apex Class in Salesforce | Apex Class for Beginners

## Introduction

An Apex Class is a template or blueprint used to create objects and define business logic in Salesforce.

Apex Classes are used to:
- Write reusable code
- Implement business logic
- Perform calculations
- Handle database operations
- Build scalable Salesforce applications

Apex is an object-oriented programming language similar to Java.

---

# What is Apex Class?

An Apex Class is a collection of:
- Variables
- Methods
- Constructors
- Business logic

Classes help developers organize code in a structured and reusable way.

---

# Example Apex Class

```apex
public class Calc {

    public double ans;

    public double add(Integer a, Integer b) {
        ans = a + b;
        return ans;
    }

    public double sub(Integer a, Integer b) {
        ans = a - b;
        return ans;
    }
}
```

---

# Explanation of the Apex Class

## Class Declaration

```apex
public class Calc
```

- `public` → Access modifier
- `class` → Keyword used to create class
- `Calc` → Class name

---

# Variable Declaration

```apex
public double ans;
```

## Explanation

- `public` → Accessible from anywhere
- `double` → Data type
- `ans` → Variable name

This variable stores the calculation result.

---

# Method: Addition

```apex
public double add(Integer a, Integer b) {
    ans = a + b;
    return ans;
}
```

## Explanation

This method:
- Accepts two integer parameters
- Adds both values
- Stores result in `ans`
- Returns the final answer

---

# Method: Subtraction

```apex
public double sub(Integer a, Integer b) {
    ans = a - b;
    return ans;
}
```

## Explanation

This method:
- Accepts two integer parameters
- Subtracts second value from first
- Stores result in `ans`
- Returns the final answer

We can execute Apex Class using:

# Execute Anonymous Code

```apex
Calc c = new Calc();

double res;

res = c.add(5,10);

System.debug('Result : ' + res);

res = c.sub(25,5);

System.debug('Sub : ' + res);
```

Open:

```text
https://login.salesforce.com
```

- Username
- Password

---

# Step 2: Open Developer Console

Click:
```text
Profile Icon
→ Developer Console
```

---

# Step 3: Create Apex Class

Click:
```text
File
→ New
→ Apex Class
```

Enter Class Name:
```text
Calc
```

---

# Step 4: Write Apex Code

Paste the Apex Class code.

---

# Step 5: Save the Class

Shortcut:
```text
Ctrl + S
```

---

# Step 6: Execute the Class

Open:
```text
Debug
→ Open Execute Anonymous Window
```

Shortcut:
```text
Ctrl + E
```

Paste Execute Anonymous code and click:
```text
Execute
```
