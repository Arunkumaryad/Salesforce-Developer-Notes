# How to Create Custom App in Salesforce Classic

# Introduction

Salesforce allows users to create custom applications according to business requirements.

A Custom App in Salesforce is a collection of:
- Tabs
- Objects
- Reports
- Dashboards
- Functionalities

Custom Apps help users organize data and work more efficiently.

---

# What is a Custom App?

## Definition

A Custom App in Salesforce is a user-defined application that groups related tabs, objects, and features together for a specific business purpose.

Example:
- School Management App
- Library Management App
- Employee Management App

---

# Advantages of Custom Apps

- Easy navigation
- Better organization
- User-friendly interface
- Faster access to objects
- Customized business solution

---

# Steps to Create Custom App in Salesforce Classic

# Step 1: Open Setup

Click on:

```text
Setup
```

from the top-right corner.

---

# Step 2: Open Apps Section

Go to:

```text
Build → Create → Apps
```

---

# Step 3: Click on New

Click the:

```text
New
```

button to create a new custom app.

---

# New Custom App Wizard

Salesforce Classic provides a 4-step wizard for creating a custom app.

---

# Step 1 of 4: Enter App Details

Fill the following information:

| Field | Example |
|-------|----------|
| App Label | Arun Tech School |
| App Name | Arun_Tech_School |
| Description | School Management Application |

---

# Explanation of Fields

## App Label
The display name of the application shown to users.

Example:
```text
Arun Tech School
```

---

## App Name
Unique API name used internally by Salesforce.

Example:
```text
Arun_Tech_School
```

---

## Description
Used to explain the purpose of the application.

Example:
```text
This application is used for school management.
```

---

After entering details click:

```text
Next
```

---

# Step 2 of 4: Choose App Logo

You can:
- Upload logo/image
- Use default logo

Click:

```text
Insert an Image
```

to upload a custom logo.

---

# Important Note About App Logo

- Logo size should be less than 20 KB
- Recommended size:
  - 300 px width
  - 55 px height

---

After selecting image click:

```text
Next
```

---

# Step 3 of 4: Choose Tabs

In this step select tabs that will appear inside the application.

---

# Available Tabs

Tabs available in Salesforce.

Examples:
- Reports
- Students
- Colleges
- Books

---

# Selected Tabs

Tabs selected for the application.

Example:
- Home
- Colleges
- Students
- Books

---

# Add and Remove Buttons

## Add
Moves tabs from Available Tabs to Selected Tabs.

## Remove
Removes tabs from Selected Tabs.

---

# Default Landing Tab

The default tab that opens when users launch the app.

Example:
```text
Home
```

---

After selecting tabs click:

```text
Next
```

---

# Step 4 of 4: Assign to Profiles

Select profiles that can access the application.

---

# Profile Visibility

## Visible
Users with this profile can access the app.

## Default
Makes the app default for selected profile users.

---

# Example Profiles

- System Administrator
- Standard User
- Marketing Profile

---

After selecting profiles click:

```text
Save
```

---

# Custom App Successfully Created

After saving, the custom app appears in the App Dropdown Menu.

Example:
```text
Arun Tech School
```

---

# Final Output

The application contains:
- Home Tab
- Colleges Tab
- Students Tab
- Books Tab

Users can now:
- Create records
- View records
- Manage business data
