# How To Create Custom App, Objects, Tabs, Fields (Columns), Records (Rows)

# Introduction

Salesforce provides different components to manage applications and data.

In Salesforce, there are two types of components:

1. Standard
2. Custom

---

# Main Salesforce Components

## 1. App
## 2. Object
## 3. Tab
## 4. Object: Field (Columns)
## 5. Object: Records (Rows)

---

# 1. What is an App in Salesforce?

## Definition

An App in Salesforce is a collection of tabs, objects, pages, and functionalities grouped together for a specific business purpose.

Apps help users:
- Access related objects
- Manage business processes
- Organize data easily

---

# Types of Apps in Salesforce

1. Standard Apps
2. Custom Apps

---

# Steps to Create Custom App in Salesforce

## Step 1: Open Setup

Go to:

```text
Setup → Apps → App Manager
```

---

## Step 2: Click on "New Lightning App"

Click:

```text
New Lightning App
```

---

## Step 3: Enter App Details

Fill the following details:

| Field | Example |
|-------|----------|
| App Name | Demo App |
| Developer Name | Demo_App |
| Description | This is My Demo App |

Then click:

```text
Next
```

---

# App Branding

You can:
- Add app logo/image
- Change app color
- Customize branding

---

# Step 4: Configure App Options

Select:
- Navigation Style
- Supported Form Factors

## Navigation Styles

### Standard Navigation
Used for normal navigation.

### Console Navigation
Used for service-based applications.

---

# Supported Form Factors

- Desktop and Phone
- Desktop
- Phone

Click:

```text
Next
```

---

# Step 5: Utility Items

You can add utility items if required.

Example:
- Notes
- History
- Softphone

Otherwise directly click:

```text
Next
```

---

# Step 6: Navigation Items

Choose objects/tabs to display inside the app.

Move required items from:
```text
Available Items
```

to:
```text
Selected Items
```

Then click:
```text
Next
```

---

# Step 7: User Profiles

Select profiles that can access this app.

Example:
```text
System Administrator
```

Click:
```text
Save & Finish
```

---

# What is an Object in Salesforce?

## Definition

An Object in Salesforce is a database table used to store records.

Objects contain:
- Fields (Columns)
- Records (Rows)

---

# Types of Objects

1. Standard Object
2. Custom Object

---

# Examples of Standard Objects

- Account
- Contact
- Opportunity
- Lead

---

# Examples of Custom Objects

- College
- Employee

# What is a Tab in Salesforce?

## Definition

A Tab is used to display an object in Salesforce UI.

Tabs help users:
- Open objects
- Access records
- Navigate easily

---

# Types of Tabs

1. Custom Object Tab
2. Web Tab
3. Visualforce Tab
4. Lightning Component Tab

---

After saving custom object:

## Step 1

Choose Tab Style.

## Step 2

Add Description (Optional).

## Step 3

Click:
```text
Next
```

## Step 4

Assign profiles.

## Step 5

Save the tab.

# What are Fields in Salesforce?

## Definition

Fields are columns inside an object used to store specific information.

Example:

| Student Object Fields |
|-----------------------|
| Name |
| Roll Number |
| Branch |
| College |
| Contact Number |

---

# Example: College Object Fields

| College Object Fields |
|-----------------------|
| Name |
| Id |
| City |
| University |
| Contact Number |

---

# Common Field Types

| Field Type | Usage |
|------------|-------|
| Text | Store text |
| Number | Store numbers |
| Date | Store dates |
| Email | Store email |
| Phone | Store phone number |
| Checkbox | True/False |

---

# Steps to Create Fields

## Step 1

Open object:
```text
Object Manager → Student
```

---

## Step 2

Open:
```text
Fields & Relationships
```

---

## Step 3

```text
New
```

---

## Step 4

Select field type.

Example:
```text
Text
Number
Phone
Email
```

---

## Step 5

Enter field details.

Example:

| Field Label | Data Type |
|-------------|------------|
| Branch | Text |
| College | Text |
| Contact Number | Number |
| Roll Number | Number |

---

# What are Records in Salesforce?

## Definition

Records are rows inside an object table.

Each record stores actual data.

---

# Example of Student Record

| Name | Roll Number | Branch | College | Contact Number |
|------|-------------|---------|----------|----------------|
| Arun | 101 | CSE | ABC College | 9876543210 |

---

# Example of College Record

| Name | Id | City | University | Contact Number |
|------|----|------|-------------|----------------|
| ABC College | C01 | Ranchi | XYZ University | 9876543210 |

---

# Relationship Between Components

| Component | Purpose |
|-----------|----------|
| App | Collection of tabs and objects |
| Object | Database table |
| Field | Column in object |
| Record | Row in object |
| Tab | UI access for object |
