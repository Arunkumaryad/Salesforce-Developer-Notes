# Use querySelectorAll() to Fetch Data in LWC | HTML Template Data Binding

## Overview

In Lightning Web Components (LWC), the `querySelectorAll()` method is used to select **multiple HTML elements** from a component template. Unlike `querySelector()`, which returns only the first matching element, `querySelectorAll()` returns **all matching elements** as a **NodeList**.

It is commonly used when you need to:

- Read values from multiple input fields.
- Perform validation on several fields.
- Reset multiple inputs.
- Apply changes to multiple elements at once.

In this tutorial, you'll learn:

- What is `querySelectorAll()`?
- CSS selectors supported by `querySelectorAll()`.
- Complete working example.
- How to fetch values from multiple input fields.
- Best practices.
- Common mistakes.
- Interview questions.

---

# What is querySelectorAll()?

According to the video:

1. **The querySelectorAll() method returns all elements that match a CSS selector(s).**

2. **The querySelectorAll() method returns a NodeList.**

3. **The querySelectorAll() method throws a SYNTAX_ERR exception if the selector(s) is invalid.**

Unlike `querySelector()`, which returns only one element, `querySelectorAll()` returns **all matching elements** inside the current component.

---

# Syntax

```javascript
this.template.querySelectorAll('CSS Selector');
```

Returns

```javascript
NodeList
```

Example

```javascript
const inputs =
    this.template.querySelectorAll('lightning-input');
```

---

# CSS Selectors Supported

As shown in the video, `querySelectorAll()` supports different CSS selectors.

## 1. By Tag

```javascript
querySelectorAll('tagname')
```

Example

```javascript
this.template.querySelectorAll('lightning-input');
```

Returns every `lightning-input` inside the component.

---

## 2. By Class Name

```javascript
querySelectorAll('.classname')
```

Example

```javascript
this.template.querySelectorAll('.inputSection');
```

Returns every element having the class `inputSection`.

---

## 3. By Tag and Class Name

```javascript
querySelectorAll('tagname.classname')
```

Example

```javascript
this.template.querySelectorAll(
    'lightning-input.inputSection'
);
```

Returns every `lightning-input` having the class `inputSection`.

---

## JavaScript File

### querySelectorAllExamples.js

```javascript
import { LightningElement } from 'lwc';

export default class QuerySelectorAllExamples extends LightningElement {

    fName = 'Arun';
    lName = 'Kumar';

    handleUserName() {

        var input =
            this.template.querySelectorAll('lightning-input');

        input.forEach(function(element) {

            if(element.name == 'firstName') {

                this.fName = element.value;

            }

            else if(element.name == 'lastName') {

                this.lName = element.value;

            }

        }, this);

    }

}
```

# Why Use querySelectorAll()?

Use `querySelectorAll()` when:

- Multiple input fields exist.
- Multiple checkboxes need to be processed.
- Validation is required for many fields.
- Multiple elements need styling.
- Multiple values need to be collected before saving.

---

# querySelector() vs querySelectorAll()

| querySelector() | querySelectorAll() |
|-----------------|--------------------|
| Returns first matching element | Returns all matching elements |
| Returns a single element | Returns a NodeList |
| Used for one element | Used for multiple elements |
| No iteration required | Usually iterated using `forEach()` |

Example

```javascript
this.template.querySelector(
    'lightning-input'
);
```

Example

```javascript
this.template.querySelectorAll(
    'lightning-input'
);
```

---

# What is a NodeList?

A **NodeList** is a collection of DOM elements returned by `querySelectorAll()`.

Example

```javascript
const inputs =
this.template.querySelectorAll(
'lightning-input'
);
```

The `inputs` variable contains every matching element.

You can loop through it using

```javascript
inputs.forEach(element => {

});
```

# Common Mistakes

## Forgetting `this.template`

Wrong

```javascript
querySelectorAll('lightning-input');
```

Correct

```javascript
this.template.querySelectorAll(
'lightning-input'
);
```

---

## Assuming It Returns One Element

Wrong

```javascript
this.template
.querySelectorAll('lightning-input')
.value;
```

Correct

```javascript
const inputs =
this.template.querySelectorAll(
'lightning-input'
);

inputs.forEach(element => {

});
```

---

## Invalid CSS Selector

Wrong

```javascript
querySelectorAll('###input');
```

This throws a syntax error because the selector is invalid.

---

## Losing the `this` Context

If using a normal function inside `forEach()`, remember to pass `this` as the second argument (as shown in the example), or use an arrow function.

Example

```javascript
inputs.forEach((element) => {

    this.fName = element.value;

});
```

---

# Best Practices

- Use `querySelectorAll()` only when multiple elements need to be accessed.
- Use meaningful class names and `name` attributes.
- Prefer arrow functions inside `forEach()` for cleaner code.
- Keep DOM manipulation minimal.
- If only one element is needed, use `querySelector()` instead.

# Interview Questions

### What is querySelectorAll()?

`querySelectorAll()` returns **all elements** that match a specified CSS selector inside the current component template.

---

### What does querySelectorAll() return?

It returns a **NodeList** containing all matching elements.

---

### Can querySelectorAll() return a single element?

Yes. If only one element matches the selector, the returned NodeList contains one element. However, the return type is still a NodeList.

---

### What is the difference between querySelector() and querySelectorAll()?

- `querySelector()` returns the first matching element.
- `querySelectorAll()` returns all matching elements as a NodeList.

---

### How do you iterate over the result of querySelectorAll()?

Using `forEach()`.

Example

```javascript
inputs.forEach(element => {

});
```

---

### How do you select elements by class?

```javascript
this.template.querySelectorAll(
'.inputSection'
);
```

---

### How do you select elements by tag?

```javascript
this.template.querySelectorAll(
'lightning-input'
);
```

---

### How do you select elements by tag and class together?

```javascript
this.template.querySelectorAll(
'lightning-input.inputSection'
);
```
