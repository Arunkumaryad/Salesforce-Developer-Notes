# Visualforce Email Template in Salesforce with Example

## Introduction

A Visualforce Email Template is an advanced email template in Salesforce that allows developers and administrators to create dynamic and highly customized emails using Visualforce markup.

Unlike Text, HTML, or Custom HTML templates, Visualforce Email Templates can access related records, display dynamic data, use complex logic, and render information from multiple Salesforce objects.

Visualforce Email Templates are commonly used when standard merge fields are not enough and business requirements require customized email content.

---

# What is a Visualforce Email Template?

## Definition

A Visualforce Email Template is an email template built using Visualforce markup that allows dynamic rendering of Salesforce data inside email messages.

It uses Visualforce tags such as:

```html
<messaging:emailTemplate>
<messaging:plainTextEmailBody>
<messaging:htmlEmailBody>
```

to generate email content dynamically.

---

# Why Use Visualforce Email Templates?

Visualforce Email Templates provide capabilities that are not available in standard email templates.

### Benefits

✅ Dynamic data rendering

✅ Access to related records

✅ Advanced merge field support

✅ Supports HTML and Plain Text emails

✅ Can display multiple child records

✅ Custom formatting and layouts

✅ Used in Apex, Workflow, Flow, and Process Automation

# Visualforce Code Example

Replace the existing code with:

```html
<messaging:emailTemplate
subject="Thank you for your Inquiry"
recipientType="User"
relatedToType="Lead">

<messaging:plainTextEmailBody>

Hello,

Lead information is as follows :

Name : {!relatedTo.Name}

Company : {!relatedTo.Company}

Lead Status : {!relatedTo.Status}

Rating : {!relatedTo.Rating}

</messaging:plainTextEmailBody>

</messaging:emailTemplate>
```

Click:

```text
Save
```

---

# Understanding the Code

## messaging:emailTemplate

Defines the email template.

Example:

```html
<messaging:emailTemplate>
```

---

## subject

Defines the email subject.

Example:

```html
subject="Thank you for your Inquiry"
```

---

## recipientType

Specifies who receives the email.

Example:

```html
recipientType="User"
```

---

## relatedToType

Specifies which object's data is available.

Example:

```html
relatedToType="Lead"
```

---

## messaging:plainTextEmailBody

Defines the email body in plain text format.

Example:

```html
<messaging:plainTextEmailBody>
```

---

## relatedTo

Provides access to the Related To record.

Example:

```html
{!relatedTo.Name}
```

Salesforce automatically replaces it with Lead Name.

# Difference Between Standard and Visualforce Email Templates

| Feature | Standard Template | Visualforce Template |
|----------|------------------|---------------------|
| Merge Fields | Yes | Yes |
| HTML Support | Yes | Yes |
| Dynamic Data | Limited | Advanced |
| Multiple Related Records | No | Yes |
| Apex Support | No | Yes |
| Custom Logic | No | Yes |
| Child Record Rendering | No | Yes |

# Interview Questions

## What is a Visualforce Email Template?

A Visualforce Email Template is an email template built using Visualforce markup that allows dynamic rendering of Salesforce data and supports advanced customization.

---

## What is recipientType?

recipientType specifies the object that receives the email.

Example:

```html
recipientType="User"
```

---

## What is relatedToType?

relatedToType specifies the object whose data is available inside the email template.

Example:

```html
relatedToType="Lead"
```

---

## What is relatedTo?

relatedTo provides access to the record specified in relatedToType.

Example:

```html
{!relatedTo.Name}
```

---

## Can Visualforce Email Templates display data from related records?

Yes. Visualforce Email Templates can display data from parent and child records using merge fields and Visualforce components.

---

## Can Visualforce Email Templates use Apex?

Yes. Visualforce Email Templates can work with Apex controllers and custom components.

# Summary

Visualforce Email Templates are the most powerful email template type available in Salesforce. They allow developers to create dynamic, personalized, and data-driven emails using Visualforce markup. In this example, a Visualforce Email Template was created with `recipientType="User"` and `relatedToType="Lead"`, and Lead information such as Name, Company, Status, and Rating was dynamically displayed using merge fields. This makes Visualforce Email Templates ideal for notifications, approvals, reports, and advanced business communication requirements.
