# Apex Control Structures Part 1 : Introduction | Salesforce Development

## Introduction

Control Structures in Apex are programming structures that control the flow of execution of statements.

They help developers:
- Make decisions
- Execute conditions
- Repeat statements
- Control program execution flow

Control structures are very important in Apex programming and Salesforce development.

---

# What are Control Structures in Apex?

## Definition

These are the structures in programming languages which control the flow of execution of statements.

Control structures decide:
- Which statement should execute
- How many times a statement should execute
- Which path the program should follow

---

# Types of Control Structures

1. Sequence Structure
2. Selection Structure
3. Loop Structure

---

# 1. Sequence Structure

## Definition

In Sequence Structure, statements execute one after another in sequential order.

Execution Flow:

```text
Start
 ↓
Action 1
 ↓
Action 2
 ↓
Action 3
 ↓
Action n
 ↓
End
```

---

# Example of Sequence Structure

```apex
Integer a = 3, b = 4;

Integer c;

c = a + b;

System.debug('Addition: ' + c);
```

---

# Explanation

The program executes step-by-step:
1. Variable declaration
2. Value assignment
3. Addition operation
4. Output display

---

# Output

```text
Addition: 7
```

---

# 2. Selection Structure

## Definition

Selection Structure is used to select one path among multiple paths based on condition.

Execution Flow:

```text
Condition → True → Action 1
Condition → False → Action 2
```

---

# How is Selection Structure implemented in Apex?

1. Simple if
2. if else
3. Nested if
4. Else if Ladder
5. switch when

---

# Simple if Statement

## Syntax

```apex
if(condition) {

    // statements
}
```

---

# if else Statement

## Syntax

```apex
if(condition) {

    // true block

} else {

    // false block
}
```

---

# Nested if Statement

## Definition

An if statement inside another if statement is called Nested if.

---

# Example

```apex
if(condition1) {

    if(condition2) {

        // statements
    }
}
```

---

# Else if Ladder

## Definition

Else if ladder is used to check multiple conditions one by one.

---

# Example

```apex
if(condition1) {

}
else if(condition2) {

}
else if(condition3) {

}
else {

}
```

---

# switch when Statement

## Definition

`switch when` is used to execute one block among multiple blocks based on matching condition.

---

# Syntax

```apex
switch on variable {

    when value1 {

    }

    when value2 {

    }

    when else {

    }
}
```

---

# Execution Flow of switch when

```text
Start
 ↓
Switch when
 ↓
Action based on matching condition
 ↓
End
```

---

# Example

```apex
Integer day = 2;

switch on day {

    when 1 {

        System.debug('Monday');
    }

    when 2 {

        System.debug('Tuesday');
    }

    when else {

        System.debug('Invalid Day');
    }
}
```

---

# Output

```text
Tuesday
```

---

# 3. Loop Structure

## Definition

Loop Structure is used to execute statements repeatedly until condition becomes false.

Execution Flow:

```text
Start
 ↓
Action
 ↓
Condition
 ↓
True → Repeat
False → End
```

---

# How is Loop Structure implemented in Apex?

1. for loop
2. while loop
3. do while loop

---

# for Loop

## Syntax

```apex
for(initialization; condition; increment/decrement) {

    // statements
}
```

---

# Example

```apex
for(Integer i = 1; i <= 5; i++) {

    System.debug(i);
}
```

---

# Output

```text
1
2
3
4
5
```

---

# while Loop

## Syntax

```apex
while(condition) {

    // statements
}
```

---

# Example

```apex
Integer i = 1;

while(i <= 5) {

    System.debug(i);

    i++;
}
```

---

# do while Loop

## Syntax

```apex
do {

    // statements

} while(condition);
```

---

# Example

```apex
Integer i = 1;

do {

    System.debug(i);

    i++;

} while(i <= 5);
```

---

# Important Difference Between Loops

| Loop | Condition Check |
|------|----------------|
| for | Before execution |
| while | Before execution |
| do while | After execution |

---
