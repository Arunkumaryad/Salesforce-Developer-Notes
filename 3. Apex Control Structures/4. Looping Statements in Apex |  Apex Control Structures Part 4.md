# Looping Statements in Apex | Apex Control Structures Part 4

## Introduction

Loop structures are used to execute a block of statements repeatedly until a condition becomes false.

Loops help:
- Reduce code repetition
- Improve efficiency
- Automate repetitive tasks

---

# What are Loop Structures?

## Definition

Loop structures are control structures that repeatedly execute a block of code while a condition is true.

Execution Flow:

```text
Start
 ↓
Execute Statements
 ↓
Check Condition
 ↓
True  → Repeat Again
False → End
```

---

# Types of Loop Structures in Apex

Apex supports:
1. for loop
2. while loop
3. do while loop

---

# Components of Loop

A loop mainly contains:

```text
initialization
condition
increment/decrement
```

---

# Working of Loop Components

## 1. Initialization
Runs only once before loop starts.

Example:
```apex
Integer i = 0;
```

---

## 2. Condition
Runs in every iteration.

If condition becomes false, loop stops.

Example:
```apex
i < 5
```

---

## 3. Increment/Decrement
Updates loop variable in every iteration.

Example:
```apex
i++
```

---

# 1. for Loop

## Syntax

```apex
for(initialization; condition; increment/decrement) {

    /* statements */

}
```

---

# Flow of for Loop

1. Initialization executes once
2. Condition checks
3. Statements execute
4. Increment/Decrement executes
5. Again condition checks

---

# Example of for Loop

```apex
/* To print Square of 0 to 4 */

for(Integer i = 0; i < 5; i++) {

    System.debug('Square of ' + i + ' = ' + i*i);

}
```

---

# Output

```text
Square of 0 = 0
Square of 1 = 1
Square of 2 = 4
Square of 3 = 9
Square of 4 = 16
```

---

# Explanation

- Loop starts from:
```apex
i = 0
```

- Runs until:
```apex
i < 5
```

- In every iteration:
```apex
i++
```

increases value by 1.

---

# 2. while Loop

## Syntax

```apex
initialization

while(condition) {

    /* statements */

    increment/decrement
}
```

---

# Flow of while Loop

1. Initialization executes
2. Condition checks
3. Statements execute
4. Increment/Decrement executes
5. Again condition checks

---

# Example of while Loop

```apex
/* To print Square of 0 to 4 */

Integer i = 0;

while(i < 5) {

    System.debug('Square of ' + i + ' = ' + i*i);

    i++;
}
```

---

# Output

```text
Square of 0 = 0
Square of 1 = 1
Square of 2 = 4
Square of 3 = 9
Square of 4 = 16
```

---

# Explanation

- Initialization is written outside loop
- Condition checks before execution
- If condition becomes false, loop stops

---

# 3. do while Loop

## Syntax

```apex
initialization

do {

    /* statements */

    increment/decrement

}
while(condition);
```

---

# Flow of do while Loop

1. Statements execute first
2. Condition checks later
3. If condition is true, loop repeats

---

# Important Point

`do while` executes at least one time even if condition is false.

---

# Example of do while Loop

```apex
/* To print Square of 0 to 4 */

Integer i = 0;

do {

    System.debug('Square of ' + i + ' = ' + i*i);

    i++;

}
while(i < 5);
```

---

# Output

```text
Square of 0 = 0
Square of 1 = 1
Square of 2 = 4
Square of 3 = 9
Square of 4 = 16
```

---

# Difference Between for, while and do while

| Loop Type | Condition Check | Best Use Case |
|-----------|-----------------|---------------|
| for | Before execution | Fixed number of iterations |
| while | Before execution | Unknown number of iterations |
| do while | After execution | Must execute at least once |
