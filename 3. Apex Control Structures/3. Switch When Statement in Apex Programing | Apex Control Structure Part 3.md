# Switch When Statement in Apex Programming | Apex Control Structure Part 3

## Introduction

`switch when` statement in Apex is a selection control structure used to execute one block among multiple blocks based on matching condition.

It makes code:
- Cleaner
- More readable
- Easier to maintain

`switch when` is mainly used when:
- Multiple conditions are checked
- One value is compared against many options

---

# What is switch when Statement?

## Definition

`switch when` is a control structure used to execute one block among multiple blocks depending on matching value.

Execution Flow:

```text
Start
 ↓
Switch when
 ↓
Matching Condition
 ↓
Execute Related Block
 ↓
End
```

---

# How is it implemented in Apex?

By using:
```text
switch when
```

---

# Syntax of switch when Statement

```apex
switch on variable {

    when value1 {

        // statements
    }

    when value2 {

        // statements
    }

    when else {

        // default block
    }
}
```

---

# Important Points About switch when

- `switch on` is used to evaluate expression
- `when` is used for matching conditions
- `when else` works like default block
- Only one matching block executes
- `break` statement is not required in Apex

---

# Example 1: switch when with Integer Expression

```apex
Switch on (1 + 2) {

    when 1 {

        System.debug('Apple');
    }

    when 2 {

        System.debug('Banana');
    }

    when 3 {

        System.debug('Cat');
    }

    when else {

        System.debug('Invalid');
    }
}
```

---

# Output

```text
Cat
```

---

# Explanation

Expression:
```apex
1 + 2
```

Result:
```text
3
```

Matching block:
```apex
when 3
```

Therefore:
```text
Cat
```

is printed.

# when else Block

## Definition

`when else` block executes when no condition matches.

It acts like:
```text
default block
```

in other programming languages.

---

# Example of when else

```apex
Integer num = 10;

Switch on num {

    when 1 {

        System.debug('One');
    }

    when 2 {

        System.debug('Two');
    }

    when else {

        System.debug('Invalid Number');
    }
}
```

---

# Output

```text
Invalid Number
```

# Difference Between if else and switch when

| if else | switch when |
|---------|-------------|
| Used for complex conditions | Used for matching values |
| Less readable for many conditions | More readable |
| Can handle ranges | Handles exact matches |
| More flexible | Simpler syntax |
