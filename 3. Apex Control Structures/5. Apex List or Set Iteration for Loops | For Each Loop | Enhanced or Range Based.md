# Apex List or Set Iteration for Loops | For Each Loop | Enhanced or Range Based

## Introduction

In Apex, `for each loop` is also known as:
- Enhanced for loop
- Range based for loop
- List or Set iteration loop

It is used to iterate over:
- Arrays
- Lists
- Sets

This loop makes code simpler and more readable.

---

# What is For Each Loop?

## Definition

For each loop is used to access elements one by one from:
- Array
- List
- Set

without using index values.

---

# Different Names of For Each Loop

In Apex, for each loop is also called:

- for each loop
- range based for loop
- enhanced for loop
- list or set iteration for loop

---

# Syntax of For Each Loop

```apex
for(variable : array/list/set) {

    /* statements */

}
```

---

# Syntax Explanation

| Part | Description |
|------|-------------|
| variable | Stores current element |
| : | Reads values one by one |
| array/list/set | Collection to iterate |

---

# Features of For Each Loop

- Simple syntax
- No index required
- Cleaner code
- Better readability
- Best for collections

---

# Example 1: Array Iteration using For Each Loop

```apex
/* Array */

Integer[] arr = new Integer[]{10,20,30,40,50};

String[] str = new String[]{'arun', 'kumar', 'Salesforce'};

for(Integer i : arr) {

    System.debug('value of i : ' + i);

}

for(String i : str) {

    System.debug('value of i : ' + i);

}
```

---

# Output

```text
value of i : 10
value of i : 20
value of i : 30
value of i : 40
value of i : 50

value of i : arun
value of i : kumar
value of i : Salesforce
```

---

# Explanation

## Integer Array

```apex
Integer[] arr = new Integer[]{10,20,30,40,50};
```

Loop reads values one by one:
- 10
- 20
- 30
- 40
- 50

---

## String Array

```apex
String[] str = new String[]{'arun', 'kumar', 'Salesforce'};
```

Loop reads:
- arun
- kumar
- Salesforce

---

# Example 2: List Iteration using For Each Loop

```apex
/* List */

List<Integer> arrList = new List<Integer>{10,20,30,40};

for(Integer i : arrList) {

    System.debug('value of i : ' + i);

}
```

---

# Output

```text
value of i : 10
value of i : 20
value of i : 30
value of i : 40
```

---

# Explanation

## List Collection

```apex
List<Integer> arrList
```

stores multiple Integer values.

For each loop accesses values one by one without using index.

---

# Example 3: Set Iteration using For Each Loop

```apex
/* Set */

Set<Integer> st = new Set<Integer>{20,10,33,33,40,31};

for(Integer i : st) {

    System.debug('value of i : ' + i);

}
```

---

# Output

```text
value of i : 20
value of i : 10
value of i : 33
value of i : 40
value of i : 31
```

---

# Explanation

## Important Point About Set

Set:
- Does not allow duplicate values
- Stores unique values only

Here:
```apex
33
```

is written twice but stored only once.

---

# Difference Between Array, List and Set

| Collection | Duplicate Allowed | Ordered |
|------------|------------------|----------|
| Array | Yes | Yes |
| List | Yes | Yes |
| Set | No | Not Guaranteed |

# Advantages of For Each Loop

- Easy to write
- Cleaner syntax
- No need of counter variable
- Less chances of errors
- Best for collections

---

# Limitations of For Each Loop

- Cannot access index directly
- Cannot modify collection size during iteration
- Less flexible than normal for loop
