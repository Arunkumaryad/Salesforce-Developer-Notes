# Apex Conditional Statements | Apex Control Structures Part 2

## Introduction

Conditional Statements in Apex are part of Selection Structure.

They help programs:
- Make decisions
- Execute specific blocks based on conditions
- Control the execution flow

Conditional statements are widely used in Salesforce development for:
- Validation
- Business logic
- Decision making
- Automation

---

# Selection Structure

## Definition

Selection Structure is used to select one execution path among multiple paths based on condition.

Execution Flow:

```text
Start
 ↓
Condition
 ↓
True  → Action 1
False → Action 2
 ↓
End
```

---

# How is Selection Structure implemented in Apex?

1. Simple if
2. if else
3. Nested if
4. Else if Ladder

---

# 1. Simple if Statement

## Definition

Simple `if` statement executes block only when condition becomes true.

If condition is false, statements inside `if` block are skipped.

---

# Syntax

```apex
if(condition) {

    // statements
}
```

---

# Example of Simple if Statement

```apex
Integer num = 50;

if(num >= 0) {

    System.debug('Number is Positive');
}
```

---

# Output

```text
Number is Positive
```

---

# Explanation

Condition:
```apex
num >= 0
```

evaluates to:
```text
true
```

Therefore:
```text
if block executes
```

---

# Example with False Condition

```apex
Integer num = -50;

if(num >= 0) {

    System.debug('Number is Positive');
}
```

---

# Output

```text
No Output
```

---

# Explanation

Condition becomes:
```text
false
```

So:
```text
if block does not execute
```

---

# 2. if else Statement

## Definition

`if else` statement executes:
- One block when condition is true
- Another block when condition is false

---

# Syntax

```apex
if(condition) {

    // true block

} else {

    // false block
}
```

---

# Example of if else Statement

```apex
Integer num = -50;

if(num >= 0) {

    System.debug('Number is Positive');

} else {

    System.debug('Number is Negative');
}
```

---

# Output

```text
Number is Negative
```

---

# Explanation

Condition:
```apex
num >= 0
```

becomes:
```text
false
```

Therefore:
```text
else block executes
```

---

# 3. Nested if Statement

## Definition

An `if` statement inside another `if` statement is called Nested if.

Nested if is used when:
- Multiple conditions need to be checked
- One condition depends on another condition

---

# Syntax

```apex
if(condition1) {

    if(condition2) {

        // statements
    }
}
```

---

# Example of Nested if Statement

```apex
Integer num = 50;

if(num >= 0) {

    if(Math.mod(num, 2) == 0) {

        System.debug('Number is Even Positive');

    } else {

        System.debug('Number is Odd Positive');
    }

} else {

    System.debug('Number is Negative');
}
```

---

# Output

```text
Number is Even Positive
```

---

# Explanation

First condition:
```apex
num >= 0
```

is:
```text
true
```

Second condition:
```apex
Math.mod(num,2) == 0
```

is also:
```text
true
```

So:
```text
Even Positive block executes
```

---

# 4. Else if Ladder

## Definition

Else if Ladder is used to check multiple conditions one after another.

When one condition becomes true:
- Corresponding block executes
- Remaining conditions are skipped

---

# Syntax

```apex
if(condition1) {

}
else if(condition2) {

}
else if(condition3) {

}
else {

}
```

---

# Example of Else if Ladder

```apex
Integer score = 55;

if(score >= 75) {

    System.debug('Distinction');

}
else if(score >= 60) {

    System.debug('First Class');

}
else if(score >= 50) {

    System.debug('Second Class');

}
else {

    System.debug('Fail');
}
```

---

# Output

```text
Second Class
```

---

# Explanation

Conditions checked:

```text
score >= 75 → false
score >= 60 → false
score >= 50 → true
```

Therefore:
```text
Second Class executes
```

# Difference Between Conditional Statements

| Statement | Usage |
|-----------|-------|
| if | Single condition |
| if else | Two conditions |
| Nested if | Multiple dependent conditions |
| Else if Ladder | Multiple independent conditions |

# Interview Questions

## What are Conditional Statements in Apex?

---

## What is Simple if Statement?
Simple if executes statements only when condition becomes true.

---

## What is difference between if and if else?
- `if` executes only true block
- `if else` executes either true block or false block

---

Nested if means an if statement inside another if statement.

---

## What is Else if Ladder?
Else if Ladder checks multiple conditions sequentially.

`Math.mod()` returns remainder after division.

```apex
Math.mod(10,2)
```

```text
0
```
