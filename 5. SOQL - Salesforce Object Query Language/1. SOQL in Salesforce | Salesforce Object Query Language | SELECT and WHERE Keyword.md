# SOQL in Salesforce | Salesforce Object Query Language | SELECT and WHERE Keyword

# Introduction

SOQL is one of the most important query languages in Salesforce used to fetch records from Salesforce Objects.

SOQL is mainly used for:

- Fetching Records
- Filtering Records
- Retrieving Object Data
- Database Operations
- Working with Apex and APIs

---

# SOQL

# Definition

SOQL stands for:

```text
Structured Query Language
```

In Salesforce:

```text
Salesforce Object Query Language
```

---

# What is SOQL?

SOQL is a query language used in Salesforce to retrieve data from Salesforce Objects.

It works similarly to SQL but is specially designed for Salesforce Objects and Records.

---

# Why Use SOQL?

SOQL helps developers and administrators to:

- Retrieve records from objects
- Filter records using conditions
- Fetch specific fields
- Work with Apex programming
- Perform database operations efficiently

---

# Features of SOQL

| Feature | Description |
|---|---|
| Object Querying | Fetch records from Salesforce Objects |
| Field Selection | Retrieve selected fields only |
| Filtering | Use WHERE condition |
| Fast Retrieval | Efficient record fetching |
| Salesforce Specific | Designed only for Salesforce |
| Apex Integration | Used with Apex Programming |

---

# Basic SOQL Query Structure

```sql
SELECT Field1, Field2 FROM ObjectName
```

---


This query fetches:

from:

```text
Student__c Object
```

# SELECT Keyword in SOQL

# Definition

SELECT keyword is used to retrieve fields from Salesforce Objects.

---

# Syntax

```sql
SELECT Field1, Field2, Field3 FROM ObjectName
```

---

# Example 1

```sql
SELECT Id, Name, Roll_Number__c, Dept__c FROM Student__c
```

This query fetches:

- Id
- Name
- Roll Number
- Department

from Student Object.

---

# Example 2

```sql
SELECT Id, LastName, FirstName, Name, Title, Company, Street, State, City, PostalCode, Country FROM Lead
```

This query fetches records from the Lead Object.

---

# Real-Time Example Used

Object Used:

```text
Student__c
```

Fields Used:

| Field |
|---|
| Id |
| Name |

# Query Example

```sql
SELECT Id, Name, School_Name__c, Age__c, Class__c FROM Student__c
```

---

# Output

This query returns all Student records with:

- Id
- Name
- School Name
- Age
- Class

---

# WHERE Keyword in SOQL

# Definition

WHERE keyword is used to filter records based on conditions.

---

# Syntax

```sql
SELECT Field1, Field2 FROM ObjectName
WHERE Field = Value
```

---

# Example 1

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Dept__c = 'CE'
```

This query fetches students whose department is:

```text
CE
```

---

# Example 2

```sql
SELECT Id, Name, School_Name__c, Age__c, Class__c
FROM Student__c
WHERE School_Name__c = 'Dtu'
```

This query returns students whose school name is:

```text
Dtu
```

---

# Example 3

```sql
SELECT Id, Name, School_Name__c, Age__c, Class__c
FROM Student__c
WHERE Age__c = 42
```

This query returns students whose age is:

```text
42
```

---

# Important Notes About WHERE Clause

| Condition Type | Example |
|---|---|
| Text Field | `WHERE Name = 'Arun'` |
| Number Field | `WHERE Age__c = 22` |
| Picklist Field | `WHERE Dept__c = 'IT'` |

# Running Query in Developer Console

# Step 1: Open Developer Console

Click:

```text
Gear Icon → Developer Console
```

---

# Step 2: Open Query Editor

Click:

```text
Query Editor Tab
```

---

# Step 3: Write SOQL Query

Example:

```sql
SELECT Id, Name, School_Name__c, Age__c, Class__c FROM Student__c
```

---

# Step 4: Execute Query

Click:

```text
Execute
```

Salesforce displays query results.

---

# NOTE : How to Write and Execute a SOQL Query Directly

```text
1. Open the Object in Developer Console using Open Source.

2. All fields available in the object will be displayed.

3. Press and hold the Ctrl key and select the fields you want.

4. Click on the Query button while holding Ctrl.

5. Salesforce automatically generates the SOQL query.

6. Click Execute to run the query.
```
