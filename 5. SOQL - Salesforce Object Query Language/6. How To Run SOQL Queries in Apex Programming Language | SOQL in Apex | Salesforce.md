# How To Run SOQL Queries in Apex Programming Language | SOQL in Apex | Salesforce

# Introduction

SOQL (Salesforce Object Query Language) can be used directly inside Apex code to retrieve records from Salesforce objects.

SOQL in Apex is used when we want to:

- Fetch records from Salesforce objects
- Process records programmatically
- Apply business logic on queried data
- Display or manipulate data in Apex Classes, Triggers, and Controllers

---

# What is SOQL in Apex?

# Definition

SOQL in Apex allows developers to execute Salesforce Object Query Language queries directly within Apex code and store the results in Apex collections such as Lists, Maps, and Sets.

---

# Why Use SOQL in Apex?

- Retrieve records from Salesforce database
- Process records using loops
- Apply business logic
- Perform calculations
- Use queried data in Apex Classes and Triggers
- Display records dynamically

---

# Syntax of SOQL in Apex

```apex
List<ObjectName> records =
[
    SELECT Field1, Field2
    FROM ObjectName
];
```

---

# Example 1 : SOQL Query Using List

```apex
List<Student1__c> std =
[
    SELECT Id,
           Name,
           Roll_Number__c,
           Contact_Number__c
    FROM Student1__c
];

for(Student1__c st : std)
{
    System.debug(
        'Name : ' + st.Name +
        ' Roll Number : ' + st.Roll_Number__c
    );
}
```

---

# Explanation

### Query Part

```apex
List<Student1__c> std =
[
    SELECT Id,
           Name,
           Roll_Number__c,
           Contact_Number__c
    FROM Student1__c
];
```

Retrieves all Student1__c records and stores them inside a List.

---

### Loop Part

```apex
for(Student1__c st : std)
{
    System.debug(
        'Name : ' + st.Name +
        ' Roll Number : ' + st.Roll_Number__c
    );
}
```

Iterates through each student record and prints Name and Roll Number in the Debug Log.

---

# Example 2 : SOQL Query Using Map

```apex
Map<Id, Student1__c> newMap =
new Map<Id, Student1__c>
(
    [
        SELECT Id,
               Name,
               Roll_Number__c,
               Contact_Number__c
        FROM Student1__c
    ]
);

for(Student1__c st : newMap.values())
{
    System.debug(
        'Name : ' + st.Name +
        ' Roll Number : ' + st.Roll_Number__c
    );
}
```

---

# Explanation

### Creating Map

```apex
Map<Id, Student1__c> newMap =
new Map<Id, Student1__c>
(
    [
        SELECT Id,
               Name,
               Roll_Number__c,
               Contact_Number__c
        FROM Student1__c
    ]
);
```

Creates a Map where:

- Key = Record Id
- Value = Student1__c Record

---

### Accessing Records

```apex
newMap.values()
```

Returns all Student records stored in the Map.

---

### Looping Through Map Values

```apex
for(Student1__c st : newMap.values())
{
    System.debug(
        'Name : ' + st.Name +
        ' Roll Number : ' + st.Roll_Number__c
    );
}
```

Displays student information from the Map.

# Difference Between List and Map

| List | Map |
|--------|--------|
| Stores records sequentially | Stores key-value pairs |
| Access using index | Access using record Id |
| Best for iteration | Best for searching records |
| Can contain duplicate values | Keys are always unique |

# Where Can SOQL Be Used in Apex?

### Apex Class

```apex
List<Account> accList =
[
    SELECT Id, Name
    FROM Account
];
```

### Trigger

```apex
List<Contact> conList =
[
    SELECT Id, Name
    FROM Contact
];
```

### Batch Apex

```apex
Database.getQueryLocator(
'SELECT Id, Name FROM Account'
);
```

### Controller Class

```apex
public List<Account> accList =
[
    SELECT Id, Name
    FROM Account
];
```

# Common Collections Used with SOQL

## List

```apex
List<Account> accList
```

Stores multiple records.

---

## Map

```apex
Map<Id, Account> accMap
```

Stores records with Id as key.

---

## Set

```apex
Set<Id> accIds
```

Stores unique values.

# Best Practices

### Query Only Required Fields

```apex
SELECT Id, Name
FROM Account
```

Avoid:

```apex
SELECT *
FROM Account
```

(SOQL does not support *).

---

### Avoid SOQL Inside Loops

Bad Practice:

```apex
for(Account acc : accList)
{
    [SELECT Id FROM Contact];
}
```

---

Good Practice:

```apex
List<Contact> conList =
[
    SELECT Id
    FROM Contact
];
```

---

### Use Collections

Use:

- List
- Map
- Set

for better performance and bulk processing.
