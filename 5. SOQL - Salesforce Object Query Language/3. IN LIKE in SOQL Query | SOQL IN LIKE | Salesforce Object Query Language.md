# IN LIKE in SOQL Query | SOQL IN LIKE | Salesforce Object Query Language

# Introduction

IN and LIKE are important operators in SOQL used for filtering records.

- IN operator is used to match multiple values.
- LIKE operator is used for partial text matching.

These operators are mainly used with the:

```text
WHERE Clause
```

---

# IN Operator in SOQL

# Definition

IN operator is used to compare a field with multiple values.

Instead of writing multiple OR conditions, we can use the IN operator.

---

# Syntax

```sql
SELECT Field1, Field2, Field3
FROM ObjectName
WHERE FieldName IN (Value1, Value2, Value3)
```

---

# Example 1

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Roll_Number__c IN (32, 37)
```

---

# Explanation

This query returns records where:

- Roll Number is 32
OR
- Roll Number is 37

---

# Example 2

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Dept__c IN ('CE', 'IT')
```

---

# Explanation

This query returns students whose department is:

- CE
OR
- IT

---

# Advantages of IN Operator

| Advantage | Description |
|---|---|
| Cleaner Query | Avoids multiple OR conditions |
| Easy Filtering | Multiple values handled easily |
| Better Readability | Query becomes shorter |
| Efficient | Useful for bulk filtering |

---

# LIKE Operator in SOQL

# Definition

LIKE keyword provides a mechanism for matching partial text strings.

It supports two wildcards:

```text
% and _
```

---

# Wildcards in LIKE

| Wildcard | Meaning |
|---|---|
| % | Occurrence of zero or more characters |
| _ | Occurrence of exactly one character |

---

# LIKE Syntax

# Starts With

```sql
SELECT Field1, Field2
FROM ObjectName
WHERE FieldName LIKE 'String%'
```

---

# Ends With

```sql
SELECT Field1, Field2
FROM ObjectName
WHERE FieldName LIKE '%String'
```

---

# Contains

```sql
SELECT Field1, Field2
FROM ObjectName
WHERE FieldName LIKE '%String%'
```

---

# Example 1

```sql
SELECT Id, FirstName, Name, Company, Status
FROM Lead
WHERE Status LIKE 'Working%'
```

---

# Explanation

This query returns records where Status starts with:

```text
Working
```

Examples:

- Working - Contacted
- Working - In Progress

---

# Example 2

```sql
SELECT Id, FirstName, Name, Company, Status
FROM Lead
WHERE Status LIKE '%Contacted'
```

---

# Explanation

This query returns records where Status ends with:

```text
Contacted
```

---

# Example 3

```sql
SELECT Id, FirstName, Name, Company, Status
FROM Lead
WHERE Status LIKE '%Not%'
```

---

# Explanation

This query returns records where Status contains:

```text
Not
```

---

# Difference Between IN and LIKE

| IN | LIKE |
|---|---|
| Used for exact multiple values | Used for partial text matching |
| Works with numbers and text | Mainly used with text fields |
| Alternative for multiple OR conditions | Supports wildcard searching |
