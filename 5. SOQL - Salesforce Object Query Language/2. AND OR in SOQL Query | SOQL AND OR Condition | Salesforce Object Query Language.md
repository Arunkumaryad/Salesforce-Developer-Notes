# AND OR in SOQL Query | SOQL AND OR Condition | Salesforce Object Query Language

# Introduction

AND and OR are logical operators used in SOQL to filter records using multiple conditions.

These operators are mainly used with the:

```text
WHERE Clause
```

They help us retrieve specific records based on one or more conditions.

---

# AND Operator in SOQL

# Definition

AND operator is used when all conditions must be TRUE.

Records are returned only if every condition matches.

---

# Syntax

```sql
SELECT Field1, Field2, Field3 FROM ObjectName
WHERE Field1 = Value1 AND Field2 = Value2
```

---

# Example 1

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Roll_Number__c = 32 AND Dept__c = 'CE'
```

---

# Explanation

This query returns only those students:

- Whose Roll Number is 32
- AND Department is CE

Both conditions must be TRUE.

---

# OR Operator in SOQL

# Definition

OR operator is used when any one condition can be TRUE.

Records are returned if at least one condition matches.

---

# Syntax

```sql
SELECT Field1, Field2, Field3 FROM ObjectName
WHERE Field1 = Value1 OR Field2 = Value2
```

---

# Example 1

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Roll_Number__c = 32 OR Roll_Number__c = 37
```

---

# Explanation

This query returns students whose:

- Roll Number is 32
OR
- Roll Number is 37

---

# Example 2

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Roll_Number__c = 32 OR Dept__c = 'ENTC'
```

---

# Explanation

This query returns students:

- Whose Roll Number is 32
OR
- Department is ENTC

---

# Example 3

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Roll_Number__c = 32 OR Dept__c = 'ENTC' OR Dept__c = 'IT'
```

---

# Explanation

This query returns students:

- Whose Roll Number is 32
OR
- Department is ENTC
OR
- Department is IT

---

# Combination of AND and OR

# Definition

We can combine AND and OR operators together in a single SOQL query.

Parentheses `()` are used for better condition grouping.

---

# Example

```sql
SELECT Id, Name, Roll_Number__c, Dept__c, Contact__c, College__c
FROM Student__c
WHERE Roll_Number__c = 32 AND (Dept__c = 'CE' OR Dept__c = 'IT')
```

---

# Explanation

This query returns students:

- Whose Roll Number is 32
AND
- Department is either CE or IT

---

# Difference Between AND and OR

| AND | OR |
|---|---|
| All conditions must be TRUE | Any one condition can be TRUE |
| Returns fewer records | Returns more records |
| Used for strict filtering | Used for flexible filtering |
